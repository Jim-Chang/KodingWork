# 如何用 Raspberry pi 參與 Folding@home？

## Abstract
如果有使用 Raspberry pi 架設伺服器，但是 CPU 大部分時間空閒，可以考慮捐出來做學術研究，本篇以 Folding@home 為例，雖然 Raspberry pi 算力遠不及 x86 CPU，但就算只有一點點，也能幫助研究人員找出癌症、阿茲海默症、亨廷頓舞蹈症、牛海綿狀腦病，以及 COVID-19 的治療方法！
