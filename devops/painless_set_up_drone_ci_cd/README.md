# 無痛架設 Drone CI/CD

## Abstract
架設 CI/CD 第一時間會想到老牌的 Jenkins，但他有一些缺點，比如說學習成本高、難以維護、頁面功能複雜等等。

後來逐漸出現 CI/CD 託管服務，如 github action、circle CI，引入 docker 的架構，同時 pipeline 撰寫門檻降低，功能直覺好用，但量多要付費。

有沒有可以白嫖、自己架設的呢？有，就是今天要介紹的 Drone！
